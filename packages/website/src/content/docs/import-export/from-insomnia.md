Nouto can import Insomnia v4/v5 JSON export files.

## Steps

1. In Insomnia, go to **Application > Export Data** and export as JSON
2. In Nouto, use the command **Import Insomnia Collection**
3. Select the exported JSON file
4. Request groups become folders; all request details are preserved
