title: Assertions & Scripts
description: Validate API responses and run pre/post-request scripts.
---

Assertions let you automatically verify that API responses meet your expectations.

## Assertions

Validate response properties such as:

- **Status code** equals, not equals, greater than, less than
- **Response body** contains, matches JSONPath, matches regex
- **Headers** contain specific values
- **Response time** is under a threshold
- **JSON Schema** validation

## Pre-request Scripts

Run JavaScript before a request is sent to dynamically set variables or modify the request.

## Post-request Scripts

Run JavaScript after a response is received to extract values, chain requests, or perform custom validation.
