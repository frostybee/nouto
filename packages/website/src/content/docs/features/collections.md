---
title: Collections
description: Organize requests into nested collections with folders, shared auth, and variables.
---

Collections let you group related requests and share configuration across them.

## Features

- **Unlimited nesting** with folders inside folders
- **Folder-level configuration**: auth, headers, and variables that apply to all child requests
- **Auth inheritance**: child requests can inherit or override parent auth settings
- **Drag and drop** to reorganize requests and folders
- **Collection runner** to execute all requests in sequence

## Folder Variables

Folders can define their own variables that override environment variables for all requests inside that folder.

## Import & Export

Collections can be exported in Sendry's native format or as Postman-compatible JSON. See the [Importing](/importing/from-postman/) section for details.
