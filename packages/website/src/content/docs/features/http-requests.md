title: HTTP Requests
description: Send HTTP requests with full control over method, URL, headers, params, body, and auth.
---

Nouto supports all standard HTTP methods with a full-featured request editor.

## Supported Methods

`GET` `POST` `PUT` `PATCH` `DELETE` `HEAD` `OPTIONS`

## Request Editor

- **URL bar** with method selector and send button
- **Query parameters** with key-value editor
- **Headers** with key-value editor and auto-complete for common headers
- **Body** supporting JSON, form-data, URL-encoded, raw text, XML, binary, and GraphQL
- **Auth** supporting None, Basic, Bearer Token, API Key, and OAuth 2.0

## Response Viewer

- Status code, response time, and size
- Pretty-printed JSON, XML, and HTML responses
- Raw response view
- Response headers
- Copy response to clipboard
