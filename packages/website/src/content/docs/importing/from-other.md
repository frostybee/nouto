## Thunder Client

Use the command **Import Thunder Client** and select your Thunder Client data directory.

## cURL

Use the command **Import from cURL** and paste your cURL command. Sendry parses it into a request.

## OpenAPI / Swagger

Use the command **Import OpenAPI Specification** and select a `.json` or `.yaml` OpenAPI spec file. Sendry creates a collection with one request per endpoint.

## HAR Files

Use the command **Import from URL** or import HAR files directly to recreate requests from network recordings.

## Sendry Native Format

Use **Import Sendry Collection** to import collections previously exported in Sendry's own format. All IDs are regenerated on import to avoid conflicts.
