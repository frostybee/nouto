---
title: From Postman
description: Import your Postman collections and environments into Sendry.
---

Sendry can import Postman Collection v2.1 format and Postman environments.

## Import a Collection

1. In Postman, export your collection as **Collection v2.1**
2. In Sendry, use the command **Import Postman Collection**
3. Select the exported JSON file
4. The collection appears in your sidebar

## Import an Environment

1. In Postman, export your environment
2. In Sendry, use the command **Import Postman Environment**
3. Select the exported JSON file

All variables, auth settings, and folder structure are preserved during import.
