title: Keyboard Shortcuts
description: Reference for all keyboard shortcuts in Nouto.
---

## Request Actions

| Shortcut | Action |
|---|---|
| `Ctrl+Enter` | Send request |
| `Ctrl+N` | New request |
| `Ctrl+S` | Save request |
| `Ctrl+Shift+N` | New collection |

## Navigation

| Shortcut | Action |
|---|---|
| `Ctrl+P` | Search requests |
| `Ctrl+,` | Open settings |

## Editor

| Shortcut | Action |
|---|---|
| `Ctrl+/` | Toggle comment |
| `Ctrl+D` | Select next occurrence |
| `Ctrl+Shift+F` | Format document |
