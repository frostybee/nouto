# Icons Placeholder

Icon files will be generated using `@tauri-apps/cli` icon generation:

```bash
npm run tauri icon path/to/icon.png
```

Required icons:
- 32x32.png
- 128x128.png
- 128x128@2x.png
- icon.icns (macOS)
- icon.ico (Windows)

For Phase 1, we'll use Tauri's default icons or generate from a simple Nouto logo.
